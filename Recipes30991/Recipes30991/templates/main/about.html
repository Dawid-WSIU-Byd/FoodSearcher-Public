About

What exactly is FoodSearcher?

FoodSearcher is a web application meant to display recipes from provided ingredients. It's very simple in use! All you have to do is to select multiple ingredients from our database. You should receive a list of all the possible recipes.

How does the search work?

The process is easy to explain. After providing all the ingredients, the app sends a request to receive a list of possible recipes. We sort the list - prioritizing the highest % ratio of owned and required ingredients. Clicking the recipe name opens it's respective page in another tab.

What information is displayed?

Examples include:

  • original recipe source - web page and license
  • preparation time - minutes total
  • cost - value in USD, text glows green when it's cheap
  • nutrition facts - list of nutrients, caloric breakdown and weight per serving

This web app also displays equipment and ingredients for every singular step.

I really liked one recipe. Where can I recommend it?

You can use our comment section to say something about the recipe. There's also a like button for each recipe. You can always unlike the recipe, if it happens to be boring or was liked on accident.
